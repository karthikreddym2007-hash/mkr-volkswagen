# Volkswagen India Homepage Clone

A front-end clone of the Volkswagen India website made using only HTML, CSS, and JavaScript.
Built this as a personal project during my 1st year of B.Tech to practice web development basics.

No frameworks. No libraries. Just plain HTML, CSS, and JS.

## What's inside

- Sticky navbar that changes color on scroll
- Full-screen hero section with fade-in animation
- 6 car model cards (Taigun, Tayron, Golf GTI, Tiguan, Virtus Sport, Virtus Chrome)
- Filter buttons to sort cars by type (SUV / Sedan / Hatchback)
- Prices listed on each card
- Why Volkswagen section
- Special Offers section
- Footer with links, contact info, and newsletter input
- Works on mobile, tablet, and desktop


## Files
├── index.html
├── style.css
├── script.js


## How to run

Download or clone the repo, then just open `index.html` in any browser. That's it.
